# Machine Learning

This repository contains beginner-friendly machine learning practice notes and
notebooks. The current notebook focuses on NumPy basics and compares simple
Python list operations with NumPy array operations.

## Contents

- `numpy.ipynb` - introduction to NumPy arrays, importing NumPy, and timing
  basic list vs array operations.

## Requirements

To run the notebook, install:

- Python 3
- Jupyter Notebook or JupyterLab
- NumPy

Install the main dependency with:

```bash
pip install numpy notebook
```

## How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Start Jupyter:

```bash
jupyter notebook
```

4. Open `numpy.ipynb` and run the cells.

## Topics Covered

- Printing output in Python
- Importing NumPy as `np`
- Creating NumPy arrays
- Comparing Python lists and NumPy arrays
- Measuring execution time with `process_time`
